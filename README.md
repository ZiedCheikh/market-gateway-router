# Market Gateway Router

## Building from Source
Market Gateway Router uses a Gradle-based build system. In the instructions below, ./gradlew is invoked from the root of the source tree and serves as a cross-platform, self-contained bootstrap mechanism for the build.
### Prerequisites
Git and the JDK21.
### Check out sources
```
$ git clone git@github.com:ZiedEcheikh/market-gateway-router.git
```
